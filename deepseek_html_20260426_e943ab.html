Sobre este Guia de Viagem

Este guia foi desenvolvido especialmente para os participantes do III Simpósio Internacional em Direitos Humanos e Cidadania (Porto) e do VIII Simpósio Internacional em Direitos Humanos Aplicados (Salamanca). Aqui você encontrará todas as informações necessárias para planejar sua viagem: opções de voos, sugestões de translados, hospedagens recomendadas e roteiro flexível para o intervalo entre os congressos.

Dica: Utilize os botões de idioma no topo para alternar entre português e espanhol. Todos os blocos são expansíveis — clique nos títulos coloridos para abrir/fechar as informações.

INTERVALO ENTRE OS CONGRESSOS · 21 a 23 de junho

Os participantes ficam livres para organizar um roteiro turístico próprio, podendo programar viagens para outras cidades ou irem direto para Salamanca. Uma opção é sair de Porto para Coimbra no dia 21/06 e de Coimbra para Salamanca no dia 23/06 (ônibus acessíveis). Abaixo as duas rotas completas.
